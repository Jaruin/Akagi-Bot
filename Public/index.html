Akagi-Bot

Akagi-Bot 赤木

A Discord bot that runs a game of riichi mahjong.

Akagi-Bot is meant to follow the rules outlined in the European Mahjong Association's 2016 rulebook. If you notice a discrepancy, please document it and open a Github issue, or better yet fix it and open a pull request.

WIP. Github

!mahjong

First argument will be game name, then it needs at least 3 users mentioned. It will start a game of mahjong for you and the first 3 mentions, and let all other mentions observe the game.

Observer commands

Usable in a mahjong channel to observe the state of the game.

dora - Displays the dora indicators currently flipped up.

prevailing - Displays the prevailing wind.

turn - Displays who's turn it is.

phase - Displays what phase the game is in.

Player Commands

Usable in a mahjong player channel to play the game.

hand - Displays your hand.

toggle - Switches between displaying text and unicode, or unicode only for tile display.

!roll

Rolls dice. Any number of XdY rolls added or subtracted from each other, plus any constants.

!tiles

Displays all unicode mahjong tiles.

!ragnarok

Deletes all channels created by this bot.
